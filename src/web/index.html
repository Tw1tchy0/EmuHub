Emulator Executables

Emulator Auto-Detection

Scans your system for known emulators — the EmuHub folder, its parent directory, Program Files, and AppData. Click Apply on any result to use it.

ROM Directories

Set the folder where ROMs are stored for each console. EmuHub watches these folders and automatically detects new games when files are added.

Other / Custom Emulators

Add any emulator that isn't in EmuHub's built-in list. Give it a name, point to its .exe, set its ROM folder, and specify which file extensions it accepts (comma-separated, e.g. .rom, .bin, .img).

SteamGridDB API Key
Get a free key at steamgriddb.com → Profile → Preferences → API → Generate API Key.
Your key is encrypted with Windows DPAPI and never stored in plain text. It is tied to your Windows user account and cannot be read by other users.
⚠ Your previously saved key could not be decrypted. This happens when EmuHub is run under a different Windows user account than the one that saved the key, or in a restricted/sandboxed environment. Please re-enter your key above.
